Welcome to my pizza ordering app!


The purpose of this project is to demonstrate the use of reactive forms and template-driven forms in Angular.

1. Ordering Pizza tab allows users to dynamically add items using a reactive form.

2. Evaluate Staff page displays staff members, and users can upvote/downvote staff's evaluation.

3. Sign Up page allows users to sign up with a template-driven form. I used some custom regular expression directives to check all inputs are valid.


Languages Used: TypeScript, HTML, CSS
Framework: Angular
Tools: Node.js, Bootstrap, NGX-Bootstrap ,Github, VS Code
